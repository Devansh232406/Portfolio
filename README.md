## 🎨 Features
- **Immersive ML Aesthetic**: Dark slate background, neon cyan/emerald glowing accents, and a custom abstract machine learning node graph.
- **Interactive Animations**: Powered by Framer Motion, featuring smooth staggered reveals, scroll-linked animations, and pulsating UI elements.
- **Fully Responsive**: Flawless layout scaling across desktop, tablet, and mobile browsers (includes a blurred glassmorphic mobile navigation menu).
- **Core Sections**:
  - **Hero**: Eye-catching introduction with dynamic background elements.
  - **About**: Highlight of my background in C++, Python, Data Structures, and Scikit-Learn.
  - **Skills**: Categorized grids detailing my technical arsenal (Programming, Classical ML, Web Technologies).
  - **Projects**: Deep dives into my featured works (e.g., Predictive Analytics, Algorithmic Trading Bots) with direct GitHub links.
  - **Contact**: Easy access "Get In Touch" section for recruiters and collaborators.

## 🛠️ Tech Stack
- **Frontend Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: Vanilla CSS (CSS3 Variables, Flexbox/Grid, Transform/Transitions, Glassmorphism)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 💻 Local Setup
To run this project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Devansh232406/portfolio.git
   cd portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173/`
